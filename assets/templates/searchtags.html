Naive Bayes Classifier

Naive Bayes classifier is considered as very straightforward and easy to implement. It is not a single algorithm but is comprised of a collection of various classification algorithms based on the Bayes theorem; a theorem which is used to describe the probability of an event. This probabilistic classifier makes use of all the all the various features present in the feature vector, analyzing them differently, that is considering them independent of each other.
